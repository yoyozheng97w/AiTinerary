Google Maps API 設定指南

⚠️ 重要提醒： 在實際使用前，您需要取得有效的 Google Maps API 金鑰並替換程式碼中的 `YOUR_API_KEY`。

步驟 1：取得 API 金鑰

  1. 前往 Google Cloud Console
  2. 建立新專案或選擇現有專案
  3. 在左側選單中選擇「API 和服務」→「程式庫」
  4. 搜尋並啟用以下 API：
    • Maps JavaScript API
    • Geocoding API
  5. 前往「API 和服務」→「憑證」
  6. 點擊「建立憑證」→「API 金鑰」

步驟 2：設定 API 金鑰限制（建議）

  1. 在憑證頁面點擊您剛建立的 API 金鑰
  2. 在「應用程式限制」中選擇「HTTP 參照網址」
  3. 添加您的網域（例如：localhost、您的網站網域）
  4. 在「API 限制」中選擇「限制金鑰」並只選擇需要的 API
  5. 點擊「儲存」

步驟 3：更新程式碼

將您的 API 金鑰替換到 index.html 文件中：

<!-- 將 YOUR_API_KEY 替換為您的實際 API 金鑰 -->
<script async defer
    src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap">
</script>

範例：

<script async defer
    src="https://maps.googleapis.com/maps/api/js?key=AIzaSyB1234567890abcdefghijklmnopqrstuvwxyz&callback=initMap">
</script>

測試

設定完成後，您可以測試以下景點：

  • 台北101
  • 阿里山
  • 日月潭
  • 太魯閣
  • 墾丁
💡 提示：
  • API 金鑰是敏感資訊，請勿公開分享
  • 建議設定 HTTP 參照網址限制以提高安全性
  • Google Maps API 有使用配額限制，請注意用量
